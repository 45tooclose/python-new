
How to generate Ethereum private key and address in local (offline) environmentI used this way when I needed to generate multiple Ethereum private keys and addresses for development.With ethereumjs-wallet, you can generate Ethereum private keys and addressed very easily.You can run the code in your local (offline) environment, of course.Install ethereumjs-wallet$ npm install ethereumjs-wallet --saveCode examplevar Wallet = require('ethereumjs-wallet');const wallet = Wallet.generate();console.log("privateKey: " + wallet.getPrivateKeyString());console.log("address: " + wallet.getAddressString());Referenceshttp://blog.playground.io/entry/2018/05/03/070436https://github.com/ethereumjs/ethereumjs-wallet
