
[Dev]DemoDay iDEXO — Já PagueiJá Paguei é uma startup de pagamento com o grande objetivo de acabar com as filas. Através do aplicativo você visualiza o que foi consumido e com apenas 1 toque faz o pagamento. Após isto o cupom fiscal é enviado para seu e-mail e você pode ir embora sem pegar fila ou esperar o garçom trazer a conta.Tiago e Flávio apresentam a arquitetura do Já PagueiO pagamento é feito através do celular, os aplicativos são hibridos, mas os leitores de QRCode são nativos, Java para Android e Swift para iOS.A parte do usuário é um conteúdo web renderizando para mobile, principalmente o pedido, o conteúdo é cacheado no celular e no servidor web.No inicio do projeto usaram IONIC, mas hoje não usam absolutamente nenhum framework, opção do CTO.Como base tecnológica usam bibliotecas JQuery, mySQL, NoSQL (Redis), Cassandra, Elastic Search e Nginx.A hospedagem fica na AWS em São Paulo, mySQL no RDS e o Redis é um cluster, assim como Cassandra e Elastic Search, arquivos estáticos ficam todos no S3 Bucket, configurado para subir a cada 6 horas.Como segurança usam token para todos os parceiros e autenticação própria, apenas o load balance é acessível externamente. Todas as transações são feitas via API Rest, nunca via aplicativo e como backup usam Snapshop uma vez por dia com backup automático da AWS.O que empolga, como próximo passo é o início de trabalho com Machine Learning em Python para entender a preferência de usuários e dar sugestões, por exemplo, sugerir para um restaurante a bebida preferida de um cliente.
