
Se publica la primera version de la API de Forty SevenQueridos amigos,Estamos felices en anunciar que hoy hemos publicado la primera versión de nuestra API. Todos los desarrolladores que planean entrar en la Tienda de Apps de Forty Seven pueden comenzar a desarrollar sus aplicaciones justo ahora!Hemos publicado 3 sitios web:Forty Seven | Desarrolladores — recurso de información web donde se puede encontrar toda la documentación necesaria, enlaces a varias herramientas y SDKs, así como también la lista de aplicación que podrían ser lanzadas en la tienda de Aplicaciones de Forty Seven en un comienzo.Forty Seven | API — tu “página de inicio”, un lugar donde podrás recibir tus llaves API (son requeridas para comunicarse con la API de Forty Seven), revisar las preguntas frecuentas y SDKs.Forty Seven | Explorador de API — página de zona de pruebas con todos los métodos API disponibles. Aquí podrás chequear y probar todas las posibilidades para la API de Forty Seven, revisar los métodos de respuesta y asegurar que todo funciona correctamente y como fue planificado.Nuestra API está lista para PSD2 (Directiva de Servicios de Pago Revisada) y cumple con la legislación de Reino Unido. Está basada en OpenBankProject y es totalmente compatible con él. Esto asegura que nuestra API estará frecuentemente actualizada, tanto por nuestro equipo como por varios desarrolladores externos. Debido a esto es que ya tenemos más de 24 aplicaciones financieras que podrían ser lanzadas en nuestra Tienda de Aplicaciones Fintech.Estaremos actualizando nuestra API en el futuro cercano y compartiremos noticias acerca de nuestros desarrollos contigo!Nos gustaría nuevamente darles las gracias por su apoyo, queridos contribuyentes y seguidores!Con su ayuda hemos logrado reunir esta comunidad grande y próspera que ahora se está convirtiendo en la base para el Banco Forty Seven.Vamos a conectar los mundos financieros!
