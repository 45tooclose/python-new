
WRIO Internet OS: User Centric Web Platform 3.0 Built in Blockchain TechnologyThe promise of emerging usage of blockchain technology in our daily lives is undeniably becoming more broader from finance, e-commerce , artificial intelligence, banking, medical and many more as it offers a public digital ledger that cannot be altered. Now a project is offering a promised of developing a User Centric Web platform that will going to use the blockchain technology they called it WRIO Internet OS.What is WRIO Internet OS and webGold Tokens?WRIO Internet OS is a Web 3.0 platform based on ethereum blockchain, it provides security, privacy, malware and and virus free to all your browsing activities. You can browse all you want using your personalized user interface theme, browsing without worrying to get infected WRIO Internet OS is an mobile application, browser and plugin that operates in a read only mode ignoring all kinds of scripts that gives every user the privacy and security while browsing. WRIO Internet OS is a combination of semantics + blockchain +smartcontracts a web 3.0 that can automatically connects and operates a global and decentralized pool, shareable, reusable and machine-readable semantic data. The mission of this platform is to provide every future user a human-machine interpreter and bring the web 2.0 to a machine readable web 3.0 this means that every individual user will going to have an easy browsing experience as everything which appears in your UI are customized according to your personality, identification and likes. As a source of funds to developed the platform the developer and the team are conducting a series of one time crowdsales wherein the Presale ( 10M tokens) funds will go directly to finance the development of the platform and the remaining raised funds will be used to support the webGold exchange rate that provides security for every crowdsale participants.
